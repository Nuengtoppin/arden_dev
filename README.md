# Arden Bootstrap (v0.1 — «болванка с душой»)
Слои: CORE → VECTOR → ASPECTROLOG → BUTLER → HAOS → INTERFACE.
Быстрый старт:
```ps1
cd D:\ae\arden
cargo run
```
Управление: WASD/Shift/Space/Ctrl; Alt+ЛКМ — вращение (курсор видим); колёсико — зум; Shift+F — скрыть курсор + точка; F2/Shift+F2 — грид/карта; F3 — координаты X|Z|Y и FPS.
